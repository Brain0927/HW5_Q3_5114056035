# 🚀 快速開始指南

## ⚡ 5 分鐘快速上手

### 選項 1️⃣：線上試用 (最簡單，推薦)

直接訪問：**https://ppt-redesign-demo.streamlit.app**

無需安裝任何東西，直接上傳 PPT，選擇風格，下載新檔案！

---

### 選項 2️⃣：本地運行

#### Windows 用戶

```batch
# 雙擊執行
run.bat
```

應用會自動在 http://localhost:8501 啟動

#### macOS / Linux 用戶

```bash
chmod +x run.sh
./run.sh
```

---

### 選項 3️⃣：手動安裝

```bash
# 1. 安裝依賴
pip install -r requirements.txt

# 2. 運行應用
streamlit run streamlit_app.py

# 3. 自動打開 http://localhost:8501
```

---

## 📞 技術支援

1. **應用無法打開**
   - 檢查網路連接
   - 嘗試用其他瀏覽器
   - 清除瀏覽器緩存

2. **本地運行出錯**
   - 確保 Python 3.8+
   - 運行 `pip install -r requirements.txt`
   - 重新運行 run.bat 或 run.sh

3. **下載失敗**
   - 檢查磁盤空間
   - 試試不同瀏覽器
   - 重新上傳 PPT
