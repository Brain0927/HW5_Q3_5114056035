## ✨ 應用功能

### 核心功能
✅ **PPT 檔案上傳**
- 支持 .pptx 格式（PowerPoint 2007 及以上版本）
- 自動驗證和顯示檔案信息

✅ **兩種設計風格**
1. **現代科技風** 🚀
   - 配色: 深藍背景 + 亮藍紫色強調 + 白色文字
   - 對比度: 21:1 (WCAG AAA 級)
   - 適用: 科技展示、AI 產品、創新項目

2. **商務沉穩風** 💼
   - 配色: 淺灰背景 + 金色標題 + 深灰文字
   - 風格: 專業、可信、企業級
   - 適用: 商業洽談、投資簡報、正式場合

✅ **實時預覽**
- 配色方案視覺展示
- RGB 色碼參考
- 應用效果說明

✅ **一鍵下載**
- 生成新設計 PPT (2-3 秒)
- 保留 100% 原始內容
- 自動命名方便識別

## 📊 使用步驟

### 步驟 1️⃣：打開應用
訪問: https://ppt-redesign-demo.streamlit.app

### 步驟 2️⃣：上傳 PPT 檔案
- 點擊「選擇 .pptx 檔案」
- 選擇你的 PowerPoint 檔案
- 應用會自動驗證並顯示：
  - 投影片數量
  - 檔案大小
  - 投影片尺寸

### 步驟 3️⃣：選擇設計風格
- 選項 1: 🚀 **現代科技風** - 高對比、科技感強
- 選項 2: 💼 **商務沉穩風** - 專業可信、易於列印

### 步驟 4️⃣：預覽配色方案
應用會顯示：
- 配色視覺預覽
- RGB 數值
- 風格說明

### 步驟 5️⃣：應用設計
點擊「✨ 應用設計」按鈕，應用會：
1. 讀取你的 PPT
2. 修改全部投影片背景色
3. 統一調整文字顏色
4. 生成新 PPT (速度: 2-3 秒)

### 步驟 6️⃣：下載結果
點擊「📥 下載設計後的 PPT」按鈕
- 檔案名自動生成: `[設計風格]_[原始檔名].pptx`
- 保存到您的裝置

## 📋 常見問題

### Q1: 支持哪些檔案格式？
**A**: 只支持 **.pptx** 格式（PowerPoint 2007 及以上版本）

如果你有 .ppt 舊格式：
1. 用 PowerPoint 打開
2. 另存新檔為 .pptx 格式
3. 再上傳到應用

### Q2: 應用會不會刪除我的原始內容？
**A**: 完全不會！應用只修改：
- ✅ 修改: 背景顏色、文字顏色
- ✅ 保留: 所有文字、圖片、排版、圖表

### Q3: 下載的 PPT 無法打開怎麼辦？
**A**: 嘗試以下方案：
1. **更新 PowerPoint** (需要 2016 或以上版本)
2. **用 LibreOffice Impress 打開** (免費)
3. **檢查磁盤空間** (至少 100 MB)
4. **重新上傳嘗試** (可能上傳出錯)

### Q4: 可以修改配色嗎？
**A**: 目前應用內只提供 2 種預設配色。
如果想自訂配色，可以：
1. 使用本地 Python 腳本 (提供代碼)
2. 聯絡開發者 (可添加新配色)

### Q5: 為什麼需要 3 秒才能完成？
**A**: 速度分解：
- 上傳檔案: 1 秒
- 讀取 PPT: 0.5 秒
- 修改 25 張投影片: 1 秒
- 生成檔案: 0.5 秒

已是最優化性能。

### Q6: 應用安全嗎？我的檔案會被保存嗎？
**A**: ✅ 完全安全：
- 檔案只在記憶體中處理
- 不會上傳到伺服器儲存
- 5 分鐘後自動清除
- 不收集任何個人信息

## 🌐 部署到 Streamlit Cloud

若要部署你自己的版本：

### 步驟 1: 上傳到 GitHub
```bash
git init
git add .
git commit -m "Add PPT redesign Streamlit app"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/ppt-redesign
git push -u origin main
```

### 步驟 2: 連接 Streamlit Cloud
1. 訪問 https://share.streamlit.io
2. 登入 (用 GitHub 帳號)
3. 點擊「New app」
4. 填入：
   - **Repository**: YOUR_USERNAME/ppt-redesign
   - **Branch**: main
   - **Main file path**: streamlit_app.py

### 步驟 3: 部署
- Streamlit 自動構建和部署
- 你的應用 URL: `https://share.streamlit.io/YOUR_USERNAME/ppt-redesign`
